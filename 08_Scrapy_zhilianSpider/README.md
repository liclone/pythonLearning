# zhilianSpider
爬取智联招聘信息
# 使用
- run_spider --- 运行爬虫
- get_url --- 用来获得一些链接的url
- url_city --- 城市的链接
- url_industry ---行业
- url_job ---工作
- 将爬取到的信息写入jobs.txt（程序中已注释）
